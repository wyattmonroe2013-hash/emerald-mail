# Emerald Account Services Package

This ZIP includes the first full Emerald Account ecosystem package.

## Included

- `index.html` — Emerald Account portal
- `services/mail/index.html` — Emerald Mail frontend
- `services/office/index.html` — Emerald Office Online foundation
- `services/drive/index.html` — Emerald Drive foundation
- `services/admin/index.html` — Emerald Admin Console foundation
- `services/browser/index.html` — Emerald Browser Sync foundation
- `worker/emerald-mail-worker.js` — current Emerald Mail Worker
- `docs/` — architecture, schema, and API roadmap

## Notes

Emerald Mail is included with the current Worker-connected frontend.

Office, Drive, Admin Console, and Browser Sync are frontend foundations using local browser storage. They are ready to be connected to a unified Emerald Account Worker API later.

# Emerald Account Design

Emerald Account is the central identity system for Emerald Mail, Emerald Office Online, Emerald Drive, Emerald Admin Console, Emerald Browser Sync, and future services.

## Role hierarchy

1. Network Administrator
2. Administrator
3. Service Administrator
4. Staff
5. User
6. Disabled

## Protected account

`wmonroe01@mail.monroecorp.org` is the protected Network Administrator.

Other administrators should not be able to disable, delete, demote, or reset it.

## Network Mail Access

Network Administrator can open any mailbox through an audited feature called Network Mail Access.

Admin mailbox viewing should default to View as Admin and should not mark another user's messages as read unless explicitly selected.
